# Anal_Cocksucker

> Lua-скрипт для **MoonLoader** (SA-MP / Arizona RP), который следит за игровым чатом и присылает уведомление в **Telegram**, когда пора бежать к компу — например, когда объявили явку.

Свернул игру и пьёшь чай, а в это время кинули клич на явку? Больше такого не будет — бот пнёт тебя в Telegram.


## Возможности

-  Читает чат сервера в реальном времени
-  Шлёт уведомление в Telegram по ключевым словам (по умолчанию `на явку`)
-  Удобное меню по команде `/pisya` (на базе mimgui)
-  Настройка токена бота и Telegram ID прямо в игре
-  Кнопка «Проверить связь» — мгновенный тест
-  Запросы идут в отдельном потоке — игра **не фризится**
-  Настройки сохраняются между сессиями
-  Вырезает цветовые коды SA-MP из текста уведомления
-  Корректно работает с кириллицей (CP1251 ↔ UTF-8)

##  Требования

- **MoonLoader** v0.026 или новее
- Клиент SA-MP на сервере, где явки приходят в чат (Arizona RP и т.п.)
- Библиотеки в `moonloader/lib` (в свежих сборках обычно уже есть):
  - [`mimgui`](https://github.com/THE-FYP/mimgui)
  - [`effil`](https://github.com/effil/effil)
  - **LuaSec** (`ssl.https`) + LuaSocket
  - `samp.events`, `encoding` (идут с MoonLoader)
  - `dkjson` и/или `cjson`
- Telegram-бот (создаётся за минуту у @BotFather)

##  Установка

1. Скачай `Anal_Cocksucker.lua` (вкладка **Releases** или **Code → Download ZIP**).
2. Положи файл в папку `moonloader` (туда же, где остальные `.lua`).
3. В игре введи `/reloadall` или перезайди.
4. Открой меню командой `/pisya` и настрой бота.

##  Настройка Telegram-бота

1. Напиши **@BotFather** → `/newbot` → получи **токен** вида `1234567890:AA...`
2. Напиши **@userinfobot** → `/start` → получи свой **числовой ID**
3. **Напиши своему новому боту любое сообщение** (или `/start`) — иначе Telegram запрещает боту писать тебе первым
4. В игре: `/pisya` → вставь токен и ID → нажми **«Проверить связь»**. Должно прийти «✅ подключён».

##  Использование

| Действие | Как |
|---|---|
| Открыть/закрыть меню | команда `/pisya` |
| Вкл/выкл слежение | галка «Следить за чатом» |
| Задать триггеры | поле «Ключевые слова», через запятую |
| Сохранить настройки | кнопка «Сохранить» |
| Проверить бота | кнопка «Проверить связь» |

Ключевые слова ищутся **без учёта регистра**. Пример: `на явку, явка, сбор, тревога`.

##  Как это работает

1. Хук `samp.events.onServerMessage` перехватывает строки чата (в кодировке CP1251).
2. Текст приводится к нижнему регистру и проверяется на ключевые слова.
3. При совпадении из строки вырезаются цветовые теги `{RRGGBB}`, текст конвертируется в UTF-8.
4. Через **Telegram Bot API** (`sendMessage`) уходит уведомление — запрос выполняется в фоновом потоке (`effil`), чтобы не было лагов.
5. Защита от дублей: одинаковый текст не отправляется чаще раза в 30 секунд.

##  FAQ / Troubleshooting

**Не приходят сообщения.**
Убедись, что написал боту `/start`, проверь токен и ID, нажми «Проверить связь».

**Ошибка `chat not found` / `Unauthorized`.**
Неверный ID или токен, либо ты не нажал Start у своего бота.

**Окно «раздулось» или открывается криво.**
Удали файл `moonloader/config/mimgui/Anal_Cocksucker.lua.ini` и перезагрузи скрипт.

**Кириллица отображается квадратами.**
Нужен mimgui со шрифтом `trebucbd.ttf` (входит в стандартную поставку).

##  Дисклеймер

Скрипт только **читает чат и присылает уведомление тебе же** в твой личный Telegram. Он не даёт игровых преимуществ, не автоматизирует действия и не вмешивается в игровой процесс. Используй на свой страх и риск и в соответствии с правилами твоего сервера.
